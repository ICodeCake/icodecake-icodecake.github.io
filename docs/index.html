.NET

.NET 是一个免费的跨平台开源开发人员平台，用于生成许多不同类型的应用。使用 .NET，可以使用多种语言、编辑器和库来构建 Web、移动、桌面、游戏和 IoT 等。

WEB

Web 是一个简要的系列，向你介绍 web 开发的实用性。包括超文本标记语言（HTML）、层叠样式表（CSS）、JavaScript（或TypeScript）和常见的Web框架。

DB

数据库是结构化信息或数据的有序集合，一般以电子形式存储在计算机系统中。主要包括常见的数据库使用与高性能的SQL语句。
