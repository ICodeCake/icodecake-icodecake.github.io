# 项目中使用的代码规范

代码可以根据是否是Hot Path分为两类：

  • 正常代码(90%) Safety > Readability > Performance

  • Hot Path(10%) Performance = Safety > Readability

  • 💯 推荐 const

  • 👍 考虑 ArraySegment

  • ❌ 禁止 ArrayList

  • 👎 避免 List.Add()
